# MNK game

игра в крестики нолики на поле MxN, при этом для победы нужно собрать K символов в ряд. Игра может быть как для двух игроков, так и для одного игрока против компьютера. В последнем случае компьютер может использовать алгоритмы искусственного интеллекта для выбора оптимальных ходов. Игра реализована в виде консольного приложения.
